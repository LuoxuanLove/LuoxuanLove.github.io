数学与计算

数论

数论主要研究整数的性质。

欧拉筛

获取所有小于 $maxn$ 的质数并存入 $primes$ 中，时间复杂度 $O(n)$。

可将列表 $vis$ 公开并以 $O(1)$ 时间快速判断质数。

素数与质因数

Miller-Rabin 素性测试

进阶的素数判定方法，快速判断指定数字是否为质数，在 long long 范围内测试时间复杂度为 $O(k\log n)$ 。

可在底数列表中添加随机数以进行更多轮试验，常用于对高精度数进行测试，此时时间复杂度是 $O(k\log^{3}n)$ ，其中 $k$ 为试验轮数。

Pollard-Rho 分解因数

Pollard-Rho 算法是一种随机化算法，可以在 $O(\sqrt p)=O(n^{1/4})$ 的期望复杂度获得一个非平凡因子（非平凡因子不一定是素因子），其中 p 是 n 的最小素因子。

有概率分解失败，退出并返回 n 本身。

最大质因数

利用 Miller-Rabin 素性测试与 Pollard-Rho 分解因数计算最大质因数，期望复杂度 $O(n^{1/4}logn)$ 。

扩展欧几里得算法

扩展欧几里得可判断方程 ax+by=m 是否有解，若有解可求出任意一组解、通解、最小整数解。

线性求逆元

以下方法可以在 $O(n)$ 线性时间内求解出指定数的逆元。

组合数学

常用公式

$\binom{n}{m} = \binom{n-1}{m-1} + \binom{n-1}{m} = \frac{n!}{m!(n-m)!}$

$\binom{n}{m} = \frac{n}{n-m} \binom{n-1}{m} = \frac{n-m+1}{m} \binom{n}{m-1} = \frac{n}{m} \binom{n-1}{m-1}$

$\sum_{i=m}^n\binom{i}{m} = \binom{n+1}{m+1}$

概率论

含绝对值的期望

计算两个整数区间上随机选取点的差的绝对值的期望。

关键公式：$\frac{1}{n^2}\sum_{i=1}^n\sum_{j=1}^n|i-j| = \frac{n^2-1}{3n}$

线性代数

异或线性基

构造方法

本方法构造的异或线性基具有如下性质：

  • 线性基没有异或和为 0 的子集。

  • 线性基中各数二进制最高位不同。

  • 如果某一位是一个元素的最高有效位，则其他元素在这一位均为 0 。

因为 bitset 不提供 < 操作符，可以使用用 base[n] 存最高有效位为n的元素。

第 k 小异或和

对于某个元素，选后异或和总是比不选异或和更大，所以最大异或和就是整个集合的异或和。

进一步地，排序后还可求出第 k 小异或和，只需按照 k 的二进制分解去选择基数即可获得第 k 小异或和。
